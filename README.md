Description du projet :
Le projet a pour objectif de concevoir et développer une application
Python de gestion de salles intégrant une interface graphique (GUI). L’application reposera
sur une base de données MySQL pour la persistance des données et adoptera une
architecture en couches afin d’assurer une séparation claire entre l’accès aux données, la
logique métier et l’interface utilisateur.
